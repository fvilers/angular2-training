About

The Nexus is a sample application built with MongoDB, Express, Angular 2, and Node.js. It has been written by Fabian Vilers, and contributed by Sylvain Guerin. The source code is available on GitHub under the MIT License.

It also serves as a demo application for MEAN and Angular 2 trainings.
